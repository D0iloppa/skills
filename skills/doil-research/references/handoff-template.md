# 핸드오프 원장 템플릿 (`_HANDOFF.md`)

이 파일 **하나만** 다음 세션에 넘기면 이어착수가 가능해야 한다.
매 단계 완료 직후 갱신한다. self-contained 로 유지(외부 MCP 없이도 읽힘).

```markdown
# [리서치명] — 핸드오프 원장

## 메타
- 지시자: (예: 차은우 대표님)
- 수행자: 서강준
- 작업공간: <research_dir 절대경로>
- 정본 산출물: <research_dir>/<주제>_조사.md
- 최종 갱신: YYYY-MM-DD HH:MM

## 목적
- 표면 지시: (지시문 그대로)
- 실제 의도: (표면 뒤의 진짜 질문 — 결론이 답해야 할 것)
- 조사 스코프: (A/B 등 확정된 초점)

## 조사질문 (정본 문서 목차와 일치)
- Q1. ...
- Q2. ...

## 단계 진행 상태
- [x] 1. (Opus) 골격 + 원장 생성 — 검증: ... ✅
- [x] 2. (Sonnet) <출처> raw 수집 — sources/<파일> — ✅
- [ ] 3. (Sonnet) <출처> raw 수집 — 검증: ...   ← **다음 착수 지점**
- [ ] N. (Opus) 종합·판단 — 검증: ...

## 다음 착수 지점 (현재 커서)
- 무엇을: (다음 단계 한 문장)
- 어떻게: (모델 선택 + 대상 URL/파일 + 저장경로)

## 수집된 raw 소스
| 식별자 | 원본 URL/파일 | 저장경로 | 상태 |
|--------|--------------|----------|------|
| S6 | https://... | sources/s6_116030.md | 수집완료 |

## 미해결 질문 / 주의
- ...
```
